# dsa-java

This repository contains my practice of **Data Structures and Algorithms using Java**.

## Why this repository?
- To improve problem-solving skills
- To understand common DSA patterns
- To prepare for coding interviews and internships

## What it contains
Problems organized **pattern-wise** such as:

- Arrays
- Two Pointers
- Sliding Window
- HashMap
- Binary Search
- Strings

Each solution includes:
- Clean Java implementation
- Time and space complexity
- Pattern used for solving the problem

I will continue adding problems as I practice and learn new patterns.
